# Store Credit Limit

This is a Shopify Theme for Store Credit Limit funtionality


## Prerequisite:

```bash
NodeJS: v22.13.0
NPM: v10.9.2
```


## Installation


👨‍💻 Clone the repository

```bash
  cd to store-credit-limit folder
  npm install

```
    
## Usage

```bash
  shopify auth logout
  npm run start

```
